# Telegram Session Manager

أداة Python متقدمة مخصصة لأتمتة إدارة واكتشاف ملفات جلسات Telegram Desktop (`tdata`) مع دعم الضغط الذكي والتعامل مع Telegram Bot API.

---

# نبذة عن المشروع

يهدف مشروع **Telegram Session Manager** إلى تقديم نموذج عملي لتطبيقات الأتمتة باستخدام Python، من خلال:

- اكتشاف مجلدات Telegram Desktop تلقائياً
- التعامل مع أنظمة التشغيل المختلفة
- ضغط الملفات المهمة بكفاءة
- التكامل مع Telegram Bot API
- إدارة الملفات والعمليات الخلفية بشكل احترافي

يعكس المشروع مفاهيم متقدمة في:

- Python Automation
- File System Analysis
- Cross-Platform Development
- Telegram API Integration
- ZIP Compression Optimization
